# TwitchStreamer

Awesome tool to

  - watch a Twitch.tv Stream
  - maybe watch another Twitch.tv Stream
  - watch them all!

### Current features

  - Choose a game from the twitch game list (top 100 currently)
  - Choose the quality
  - Settings are being saved
  - See your followed streams
  - Hiding the livestreamer console
